# 个人中心与成就系统模块交接文档

## 模块说明
个人中心模块展示了用户的核心学习数据，包括等级、学习天数、学习时长统计、知识点掌握度以及收集的勋章。

## 已实现功能
- `pages/profile/index.uvue`：重构了个人中心页面结构。
- 接入了 Mock 后端数据，实现了四种核心勋章（三宅同辉/九章规制/紫宸经纬/飞虹贯宇）的展示。
- 新增了“知识点掌握度”多维度展示（由于图表库在 uni-app x 中的兼容性问题，暂时降级为进度条展示各个维度的掌握百分比）。
- 数据与全局状态 `store/user.uts` 联动，在未登录状态下不显示用户数据，登录后自动拉取并展示。
