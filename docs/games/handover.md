# 游戏式沉浸学习模块交接文档

## 模块说明
通过游戏化的方式（拼图、消消乐、扫雷等），提升用户学习古建筑知识（如榫卯结构、瓦当纹样）的趣味性与沉浸感。

## 已实现功能
- `pages/game/index.uvue`：搭建了游戏大厅入口列表，包含斗拱拼图、瓦当消消乐、古建扫雷三个主要游戏模块入口。
- 图谱首页 (`pages/learn/index.uvue`) 现已支持跳转至游戏大厅。
- 点击具体的游戏由于暂未实现原生引擎逻辑，目前降级为 Toast 提示开发中。
