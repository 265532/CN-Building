# AI 导学模块交接文档

## 模块说明
提供基于大语言模型的上下文对话能力，并支持用户上传照片进行古建筑视觉识别和结构解析。

## 已实现功能
- `pages/ai/index.uvue`：聊天界面交互优化，支持发送文字与图片。
- 接入了 `uni.chooseImage`，支持用户从相册选择或拍照上传建筑图片。
- `services/ai.uts`：增加了对 `imageUrl` 参数的支持。当前处于降级状态，当带有图片时，Mock 服务会返回视觉识别降级提示（模拟明清官式建筑构件解析）。
- 支持快速提问快捷键（知识解析、拍照识别联动）。
