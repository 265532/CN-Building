  - **交付物**: 更新 `pages/profile/index.uvue` 及相关状态流转，编写 `docs/profile/handover.md`。
- [x] **Task 2.2 - 智能导览与地图服务 API 对接**

- [x] **Task 2.3 - AI 导学模块真实对接 (含视觉识别)**

  - **描述**: 搭建知识图谱模块（建筑网络、时空图谱、技术演化）与跨文化对照模块（中西对比、符号解析、时间轴对照），支持数据的动态拉取与呈现。

  - **描述**: 开发基于原生组件或 Canvas 的游戏式交互模块，包含建筑拼图、建筑消消乐、建筑扫雷。
