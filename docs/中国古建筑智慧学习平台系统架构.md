# 中国古建筑智慧学习平台系统架构

## 1. 小程序端功能模块

### 1.1 用户系统

#### 1.1.1 个人中心

**基本信息**

- 头像
- 昵称
- 国籍认证
  **学习记录**
- **已学习中文天数**：自注册日起连续计数。
- **知识点掌握度**：以雷达图呈现五大维度。
  - 民居结构：85%
  - 官府礼制：72%
  - 皇宫装饰：64%
  - 桥梁技术：91%
  - 跨文化对比：58%
- **学习时长统计**：按日/周/月展示。
  - 示例：VR参观故宫（42分钟）
  - 示例：赵州桥AI动画解析（18分钟）
  - 示例：徽派建筑3D拆解（27分钟）
    **账户设置**
- 提供基础账户配置功能。

#### 1.1.2 权限管理

**用户角色**

- 游客
- 注册用户

#### 1.1.3 成就系统

**学习勋章**

- **民居勋章（“三宅同辉”青铜徽章）**：
  - 获取条件：完成四合院（北京）、徽派马头墙（安徽宏村）、客家土楼（福建永定）三大单元学习。
- **官府勋章（“九章规制”银质徽章）**：
  - 获取条件：通关王府（沈阳故宫东路）、县衙（河南内乡）、总督府（南京两江总督署）案例。
- **皇宫勋章（“紫宸经纬”金质徽章）**：
  - 获取条件：掌握前朝后寝布局、太和殿斗拱层级、乾清宫藻井工艺。
- **桥梁勋章（“飞虹贯宇”钻石徽章）**：
  - 获取条件：理解赵州桥空腹拱原理、卢沟桥联拱结构、程阳风雨桥木石结合技艺。
    **学习等级**
- **Lv1 青砖**：0–20小时
- **Lv5 飞檐**：100–200小时
- **Lv10 藻井**：500+小时
  **排行榜**
- **地区榜**
- **全球榜**：
  - TOP10展示案例：“法国用户Luc Dubois完成全部榫卯演化图谱学习（127节点）”、“日本用户山田美咲创建142条UGC建筑故事”。

#### 1.1.4 多语言支持

- 支持多语言切换以服务国际用户。
