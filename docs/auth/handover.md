# 认证模块交接文档

## 模块说明
负责用户的注册与登录，支持账号密码登录以及微信一键登录（原生能力）。

## 已实现功能
- `pages/auth/login.uvue`：完整独立的登录页面，包含账号密码输入框与登录按钮。
- 集成了 `uni.login` 与自定义账号密码的登录入口。
- 将 Token 保存至全局状态并持久化至本地缓存 (`uni.setStorageSync`)。
- 当真实后端接口不可用时，具备自动降级为生成 Mock Token 的能力。
