# 智能导览与地图服务交接文档

## 模块说明
地图模块结合真实地理位置，为用户展示附近的古建筑信息，并支持按建筑类型（如民居、官府、皇宫、园林等）进行专题路径筛选。

## 已实现功能
- `pages/map/index.uvue`：接入了 `uni.getLocation` 获取真实经纬度。
- 支持失败降级：定位失败时自动降级使用默认坐标（北京故宫附近）。
- 增加了顶部横向滚动的专题分类筛选（全部分类、民居、官府、皇宫、园林）。
- `services/map.uts` 中 `fetchMapMarkers` 支持接收 `category` 参数并传给后端。当前由于缺少真实数据源，使用 Mock 接口并根据分类返回带有后缀的降级数据。
